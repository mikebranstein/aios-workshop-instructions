# Glossary

Agentic OS
- A repeatable system of agents, state, gates, and loops that delivers outcomes reliably.

Agent Contract
- A strict definition of an agent's inputs, outputs, guardrails, and done criteria.

ESS (Executable System Spec)
- A structured, machine-usable spec that agents can execute and validate.

Gate
- A required checkpoint with objective pass/fail criteria.

Loop
- A bounded retry cycle used to resolve failures before progressing.

Orchestrator
- The decision layer that routes work among agents and enforces policy.

State Machine
- Allowed states and transitions for each work item.

Evidence
- Verifiable outputs: CI checks, logs, QA results, design docs, approvals.
