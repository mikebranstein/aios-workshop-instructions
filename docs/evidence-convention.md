# Evidence Convention (Docs 2)

## Required links at closure
- ESS
- PR
- Verification evidence
- QA report

## Decision log format
- Date/time
- Gate
- Decision PASS|FAIL|BLOCKED
- Summary reason
- Next action

## Naming guidance
- ESS: docs/ess-issue-<id>.md
- QA report: docs/qa-report-issue-<id>.md
