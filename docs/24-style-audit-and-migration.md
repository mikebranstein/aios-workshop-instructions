## Style fingerprint extracted from reference workshop

1. Conversational teacher voice
- uses direct coaching language (let's, now, you should see)

2. Teach before command
- explains what and why before step execution

3. Exercise blocks
- clearly labeled exercises with start/end markers

4. Observable checkpoints
- frequent "you should see" outcomes

5. Pragmatic callouts
- NOTE, RULE, DEFINITION used to reduce ambiguity

6. Incremental scaffolding
- concept -> small exercise -> validation -> next step
